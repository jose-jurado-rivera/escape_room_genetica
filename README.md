# escape_room_genetica
Aplicació educativa interactiva sobre genètica i identificació d’espècies. Inclou proves inicials, sopa de lletres, comparació de seqüències d’ADN i activitats gamificades per desenvolupar el pensament científic i l’anàlisi de dades.
